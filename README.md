# javaGameEngine

A game engine made by Java. It allow you make game with Java. (Why do I write this? Because my computer can't run the Unity, It will be blue Screen when I open it)

# How to start

Download these file and open it in IDE, Add two process for /build/base/initWorkspace and /workspace/main/Config

Then, enjoy your coding time!

# What can it do now?

It able to draw the sprites on the window, listening to some base event now.
